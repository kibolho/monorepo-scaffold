# Design System

## Getting Started


### Installation

```bash
yarn add @kibolho/design-system
```

### Usage

```javascript
import '@kibolho/design-system/dist/index.css';
import { Button } from "@kibolho/design-system";

const App = () => {
  return (
    <div>
      <Button>Click Me</Button>
    </div>
  );
};
```
